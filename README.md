# Hindi-Handwritten-Character-Recognition
This project focuses on recognizing handwritten Hindi (Devanagari) characters using machine learning and deep learning techniques. The dataset used for training and testing the model is sourced from the UCI Machine Learning Repository. Since the dataset was very large, it could not be uploaded to GitHub and is instead provided through a Google Drive link.

Dataset Information
Original Dataset:
https://archive.ics.uci.edu/ml/datasets/Devanagari+Handwritten+Character+Dataset
The dataset was originally stored in a directory structure. After downloading it locally, the images were converted into CSV files for easier processing. These CSV files have been uploaded to Google Drive.
Google Drive Dataset Link:
https://drive.google.com/drive/folders/1oIbkgV0nVchCfysMi5wZ0XrK-6C3tc2q?usp=sharing

Project Overview
Preprocessing handwritten Devanagari character images and converting them to CSV format.
Training a deep learning model to classify Hindi characters.
Performing normalization, reshaping, and other preprocessing steps.
Evaluating model accuracy and performance metrics.

Technologies Used
Python
NumPy, Pandas
TensorFlow / Keras
Matplotlib
Google Colab
Google Drive for dataset storage

How to Use
Download the dataset from the Google Drive link.
Load the CSV files in your environment (Colab or local).
Run the training notebook or script to build the model.
Test the model with new handwritten samples.
